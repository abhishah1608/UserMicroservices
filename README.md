# UserMicroservices

Implemented Java Spring boot application using SQL server as database, and clean architecture (Repository pattern).

Created microservices for the user API service, that can create new users, authenticate existing users and generate jwt token. 

Implement prometheus, grafana and actuators for monitoring system.

Implement instrumentation to measure latency, api hit counts.

Implement multi-stage docker image generation.

Implement docker compose to integrate API application with prometheus, grafana, and other monitoring systems.
